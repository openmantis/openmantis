# Roadmap

This roadmap outlines planned milestones for OpenMantis.

## Planned Milestones

### Near Term

- CLI commands for run/status/cache management
- Basic caching layer with configurable storage location
- Prompt tokenization utilities and context budgeting helpers
- Contributor docs, templates, and CI for repeatable OSS maintenance

### Mid Term

- Runtime adapters with a consistent interface (Ollama, Foundry Local)
- Context directory indexing to reuse relevant information
- Improve cache hit rate with prompt normalization
- Optional integration tests for live local runtimes

### Longer Term

- Additional runtime integrations
- Performance profiling tools for low-spec devices
- Expanded test coverage and documentation
- Release automation and versioned changelog entries
