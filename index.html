Local model orchestration

Route, cache, and budget local AI requests.

OpenMantis is a small Node.js tool and SDK that sits between your app and local runtimes like Ollama or Foundry Local. It trims repeat work with caching, keeps prompts within token budgets, and reuses context from local files when you need it.

Features

  • Runtime adapters for local model servers.
  • Caching to avoid recomputation across sessions.
  • Token estimation and prompt budgeting helpers.
  • Directory indexing and lightweight memory storage.

Quick start

Use the CLI locally or import the package from Node.js.

npm test
node bin/openmantis.js run "hello world"

Architecture

Requests flow through a router that normalizes messages, checks the token budget, looks up a cache key, and then dispatches to a runtime adapter only when needed.

The runtime adapters speak the same high-level chat interface so additional backends can be added without changing the CLI shape.
