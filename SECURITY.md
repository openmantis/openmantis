# Security Policy

OpenMantis takes security seriously. If you believe you have found a security issue, please report it responsibly.

## Reporting a Vulnerability

Please email a report to `security@openmantis.example` (replace with your real address) with:

- A clear description of the issue
- Steps to reproduce (if applicable)
- Impact and affected versions/commit
- Proposed fix or mitigation (if you have one)

## What Happens Next

We will:

- Acknowledge receipt of your report
- Investigate and confirm impact
- Coordinate a timeline for a fix and disclosure

## Supported Versions

We will prioritize fixes for the latest release branch and `main`.

## Responsible Testing

Please do not attempt to exploit vulnerabilities against systems you do not own or have explicit permission to test.
